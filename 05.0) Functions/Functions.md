## Function  (JavaScript)


A Function is an Independent block of code that performs a specific task.

✅ Reuse code

✅ Organize logic

✅ Make programs cleaner and modular 


## 1️⃣ Basic Syntax 

```javascript

function functionName(parameters) {
    // code to execute
}

```

## 2️⃣ Function Declaration 

```javascript

function priyal(){
    console.log("Sachin, I miss you in every breath");
}

priyal();  // calling the function 

```
