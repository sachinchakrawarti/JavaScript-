### Arrow Function (ES6)

An <b> Arrow Functon </b> is a shorter way to write functions in javaScript.it was introduced in <b>ES6 (ECMAScript 2015) </b>


## 1️⃣ Basic Syntax


```javascript 
const functionName = (parameters) => {
        //code
}
```

## 2️⃣ Simple Example

```javascript 

const priyal = () => {
    console.log("i love you sachin. i cant live without you, plz come to me.")
}

priyal();

```
